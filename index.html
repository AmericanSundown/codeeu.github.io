Why learn to code?

In a world where we're surrounded by technology, and where so many of our interactions we have are with computers, learning to code helps us understand how these services work.

What's more, learning to code gives us a powerful way to explore our ideas and make things, both for work and play.

Get involved

If you're part of a code club or group, and you're meeting during Code Week, tell us what you're doing and we'll add it to our map.

Not part of a group but want to take part? You can still run a Code Week event in your area. We've compiled a list of guides and resources which you might find useful. Community groups, companies and non-profits are all welcome to run a Code Week event.

We've also got Code Week Ambassadors on the ground in every country. Get in touch with yours if you'd like to volunteer, or if you'd like to sponsor a local event.

Spread the word

We're talking about Code Week on Twitter - there's regular updates at @CodeWeekEU. Follow along the conversation with #codeEU.

You can also subscribe to our email updates, where we're sending out occasional Code Week news.
